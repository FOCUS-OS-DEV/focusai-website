# Copywriting Frameworks — Focus AI

## PAS — Problem, Agitate, Solution (Primary framework)

Best for: hero sections, pain-point sections, warm audiences who feel the pain.

```
P — Problem:  Name the exact, specific situation. Not abstract.
A — Agitate:  Consequence. What does the reader lose if nothing changes?
S — Solution: The product as logical next step (no hard sell).
+ Outcome:    Life after the solution.
```

Example (Academy):
- P: "סיימת קורס. עדיין לא בנית כלום שמישהו משתמש בו."
- A: "חודשים עברו. הרעיון עדיין רק ברקע."
- S: "Bot-Camp מלמד לבנות — לא ללמוד על בנייה."
- O: "תצא עם אפליקציה חיה, לא תיעוד."

MUST: Pain = concrete situation, not abstract category.

---

## AIDA — Attention, Interest, Desire, Action

Best for: hero sections (cold traffic), ads, email.

```
A — Attention: Bold claim / provocative question / shocking stat
I — Interest:  Proof it's relevant ("אנחנו, שבנינו לארגונים אמיתיים")
D — Desire:    Show life after transformation
A — Action:    One clear CTA
```

## StoryBrand

**Critical rule: You are Yoda. Customer is Luke. Never start with "We/Our."**

| Element | Focus AI Application |
|---------|---------------------|
| Character's goal | "רוצה להיות רלוונטי בעידן ה-AI" |
| External problem | "לא יודע לבנות מוצר בעצמו" |
| Internal problem | "מרגיש שעובר עליו הרכבת" |
| Guide (us) | "עברנו את זה. לימדנו 1,200+ איש לצאת מהצד השני." |
| The plan | "הצטרף → תרגל עם כלים אמיתיים → צא עם מוצר" |
| Success | "אחרי 120 שעות: אפליקציה חיה, סוכן AI, ביטחון מקצועי" |
| Failure stakes | "ארגונים שלא יאמצו AI עד סוף 2026 — ייפלו מאחור. לא כי בחרו. כי המתינו." |

## Jobs to Be Done (JTBD)

Customers "hire" products to make progress in a specific struggling moment.

**Old (features):** "8 מודולים, 24 שיעורים"
**JTBD:** "Bot-Camp נועד לאדם שרוצה לעזוב ישיבות ויצר משהו — ולא יודע איך להתחיל."

Four forces to address:
1. **Push** (frustration) — agitate in PAS
2. **Pull** (attraction) — desire in AIDA
3. **Anxiety** (what if it fails?) — testimonials, FAQ, guarantee
4. **Habit** (keep doing it old way) — make cost of inaction explicit

## Headline Formulas

```
A — Specific Promise:
"[עשה X] תוך [זמן] — גם אם [התנגדות]"
→ "בנה סוכן AI תוך יום אחד — גם בלי שורת קוד אחת"

B — Bold Claim:
"[מספר] [אנשים] [תוצאה ספציפית]"
→ "1,200 מנהלים בנו AI בעצמם. אתה הבא?"

C — Question:
"כמה זמן עוד [מצב כואב]?"
→ "כמה שעות עוד הצוות שלך יבזבז על דוחות שה-AI יכול לכתוב?"

D — Before/After:
"[לפני] → [אחרי]"
→ "לפני Bot-Camp: קוראים על AI. אחרי: בונים איתו."

E — Stakes:
"[זמן]. [סיכון]. [קריאה]."
→ "2026. AI כבר בארגונים שלידך. האם אתה מוכן?"
```
