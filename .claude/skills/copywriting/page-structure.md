# Page Structure — Focus AI Copywriting

## Landing Page Order (Course Pages)

```
1.  HERO              → Who + What they get + How fast/specifically
2.  PROBLEM (PAS P+A) → Name exact pain × 3–5 concrete situations
3.  SOLUTION (PAS S)  → 2 sentences + "This is for you if..." list
4.  EARLY PROOF       → 1–2 specific testimonials (NOT only at the bottom)
5.  CURRICULUM        → What you BUILD, not what you study
6.  WHO THIS IS FOR   → Explicit "זה לך אם..." / "זה לא לך אם..."
7.  INSTRUCTORS       → "Why I built this" story — not a CV
8.  TRANSFORMATION    → Before → After framing
9.  TESTIMONIALS      → Mix: career outcome + specific ROI + emotional
10. PRICING           → After full value established. Anchor + guarantee.
11. FAQ               → Real objections: "אין לי זמן" / "יקר" / "לא טכני"
12. FINAL CTA         → Repeat + what they lose by waiting
```

**Rule: Minimum 3 CTA placements per long page.**

## Social Proof Placement

Place testimonials throughout — not only at the bottom:

| Location | What to place |
|----------|--------------|
| Below hero | 1 stat: "1,200+ בוגרים מ-300 חברות" |
| After pain section | 1 testimonial matching exact pain described |
| Next to CTA | Short quote + name + role |
| Alongside pricing | 1–2 testimonials to reduce buyer's remorse |
| In FAQ | "That's exactly what X thought — here's what happened" |

### Testimonials that convert:
- Specific metric: "חסכתי 8 שעות בשבוע"
- Named person + role: "דנה כ., סמנכ"לית שיווק, Wix"
- Before/during/after in 3 sentences

### Testimonials that kill trust:
- "קורס מדהים, ממליץ לכולם!" — zero specificity = zero trust
- Anonymous, no role, no company

## B2B Consulting Pages (Different Rules)

**Quantify everything:**
- "חסוך זמן" → "הפחת 40% עלות תפעולית של אנליסטים"
- "ייעוץ מקצועי" → "ROI מוכח תוך 30 יום — או ממשיכים ללא תשלום"

**Hero formula:**
`[Outcome] for [Who] — [How/Mechanism] — [Timeframe]`
→ "ארגונים שעובדים איתנו מטמיעים AI תוך 90 יום ורואים תוצאות מדידות."

**Position as partner, not vendor:**
→ "אנחנו לא יועצים שמגיעים לכמה ימים ונעלמים. אנחנו עובדים לצידך."

## Anti-Patterns — 7 Fatal Errors

1. **Feature dumping**: "8 מודולים, 40 שיעורים" → "אחרי מודול 1: תבנה את הסוכן הראשון."
2. **Passive voice**: "יכולים להיות מיושמים" → "תשתמש בכלים תוך 48 שעות."
3. **Vague value prop**: "Solutions for Growth" → use Specificity Formula.
4. **Hedging**: אולי / ייתכן / בדרך כלל → delete.
5. **Starting with "We"**: "אנחנו ב-Focus AI..." → start with customer's world.
6. **Jargon without translation** (B2C): "RAG, fine-tuning" → always add plain-language.
7. **One CTA at bottom**: add after hero, curriculum, testimonials, and bottom.
