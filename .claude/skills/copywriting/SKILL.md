# Focus AI — Brand Writing Guide

## Core Voice

Practitioner speaking to peer — not professor lecturing, not salesperson pitching.
Think: senior developer explaining to a colleague, not a brochure.

## The Four Pillars

| Pillar | Rule |
|--------|------|
| ישיר (Direct) | Say the thing. First sentence = the point. |
| נקי (Fluff-free) | Active voice. Cut every word that carries no meaning. |
| אנושי (Human) | Peer-to-peer register. Light wit OK. Forced humor not. |
| מוכח (Proof-backed) | Replace every superlative with a number or a name. |

## Banned Words / Patterns

- **Hebrew buzzwords**: פורץ דרך, חדשני, פתרונות מתקדמים, הוליסטי, סינרגיה, מנוף, אולי, ייתכן, בדרך כלל
- **English (translated)**: "unlock your potential", "empower your journey", "transform your life"
- **Structure**: Starting with "We/Our/אנחנו/אצלנו" — start with the customer's situation instead
- **Grammar**: Passive voice ("יכולים להיות מיושמים")
- **Punctuation**: Em-dash (—) in headlines → use a period instead. Stronger punch.
- **Vague endings**: "ועוד הרבה מעבר" / "עד הסוף" / "ועוד" → replace with specific number or outcome
- **Israeli CTA trap**: "לפרטים והרשמה" — the most common LP button in Israel, lowest converting. Use outcome verb instead.
- **CV bios**: Listing credentials/titles ("מרצה ב-X, מומחה ב-Y") → tell the practitioner story instead ("בנה X לחברות ישראליות, ועכשיו מלמד אותך בדיוק איך")

## CTA Rules

| Weak | Strong |
|------|--------|
| שלח | שמור לי מקום |
| הירשם | התחל את ה-Bot-Camp שלי |
| צור קשר | קבע שיחת ייעוץ חינם |
| לחץ כאן | קבל גישה מיידית |

Always add friction-reducer below button: "ללא התחייבות" / "מלא 2 פרטים בלבד" / "X מקומות נשארו"

## Register by Audience

| Audience | Tone | Person | Verbs |
|----------|------|--------|-------|
| Academy (B2C) | Direct-warm | אתה/את | build, create, learn, become |
| Consulting (B2B) | Direct-authoritative | אתם | implement, deploy, measure, scale |

## Quick Checklist

Before publishing any copy:
- [ ] First sentence: specific situation, not abstract category?
- [ ] Any hedging language? (אולי/ייתכן) → delete
- [ ] Passive voice anywhere? → fix
- [ ] Any superlative without proof? → replace with number
- [ ] Starts with "We/Our/אנחנו/אצלנו"? → rewrite to start with customer
- [ ] CTA describes outcome, not action? (never "לפרטים והרשמה")
- [ ] Friction-reducer below every CTA button?
- [ ] Section headers tell a story when read alone?
- [ ] Em-dashes (—) in headlines? → replace with period
- [ ] Vague endings ("ועוד הרבה", "עד הסוף")? → replace with specific number/outcome
- [ ] Instructor/team bios: practitioner story or CV list? → must be story
- [ ] Year/dates: 2025 → 2026? (check footer, pricing, dates)
