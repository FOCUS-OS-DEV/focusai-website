## הפיצ'ר שכולנו חיכינו לו הגיע

ב-25 בפברואר Anthropic השיקה את **Remote Control** ל-Claude Code, ומי שמשתמש בכלי הזה ביום יום מבין למה זה עניין גדול: עכשיו אפשר לשלוט על סשן Claude Code שרץ על המחשב שלכם, ישירות מהטלפון. או מהטאבלט. או מכל דפדפן.

בלי SSH. בלי VPN. בלי פתיחת פורטים. בלי Tailscale ו-Tmux ו-Termux וכל ה-hacks שמפתחים בנו לעצמם עד עכשיו. פשוט מריצים פקודה, סורקים QR, ומתחילים לעבוד.

## איך זה עובד בפועל

ההפעלה פשוטה. מהטרמינל:

```
claude remote-control
```

או מתוך סשן קיים:

```
/rc
```

מרגע שמפעילים, Claude Code מציג URL ייחודי ו-QR code שאפשר לסרוק מהנייד. הסשן מופיע גם באפליקציית Claude (עם אייקון מחשב ונקודה ירוקה) ובממשק claude.ai/code.

הנקודה הכי חשובה: **הכל רץ מקומית.** הקבצים שלכם, שרתי ה-MCP, ה-tools, הגדרות הפרויקט, לא עוברים לענן. הטלפון הוא רק חלון לתוך הסשן המקומי. התקשורת עוברת דרך ה-API של Anthropic ב-HTTPS מוצפן, בלי פורטים נכנסים.

## מה אפשר לעשות מהנייד

- **לשלוח פרומפטים** שרצים על המחשב שלכם
- **לאשר או לדחות פעולות** (עריכת קבצים, הרצת פקודות)
- **לקרוא פלט בזמן אמת** של כל מה ש-Claude עושה
- **לעבוד ממספר מכשירים** במקביל, השיחה מסונכרנת

אם המחשב נכנס לשינה, הסשן מתחבר מחדש אוטומטית כשהוא מתעורר. אם הרשת נופלת ליותר מעשר דקות, הסשן נסגר.

פרט חשוב שכדאי לדעת: **מצב skip-permissions לא נתמך ב-Remote Control.** כלומר כל פעולה דורשת אישור ידני. מבחינת אבטחה זה הגיוני: אתם שולטים על סוכן AI שיש לו גישה מלאה למערכת הקבצים שלכם. אישור ידני מהנייד זו רשת ביטחון סבירה.

## מה צריך כדי להתחיל

- **מנוי Max** (100 עד 200 דולר לחודש). תמיכה ב-Pro (20 דולר) צפויה בקרוב
- **התחברות דרך claude.ai**, לא דרך API key
- Claude Code מותקן על המחשב (כמובן)
- אפליקציית Claude על הנייד (iOS / Android), או כל דפדפן

נכון לעכשיו הפיצ'ר לא זמין למנויי Team או Enterprise, מה שמגביל את השימוש בארגונים.

## למה זה משנה

עד עכשיו, מפתחים שרצו גישה מרחוק ל-Claude Code בנו פתרונות עקומים: SSH מ-Termux, סשנים ב-Tmux, מנהור דרך Tailscale. Remote Control מחליף את כל זה בפיצ'ר רשמי שעובד מהקופסה.

ומעבר לנוחות הטכנית, יש פה שינוי תפיסתי. הכלי הזה הופך Claude Code מ"משהו שאני מריץ כשאני יושב מול המחשב" ל"סוכן שעובד בשבילי גם כשאני לא שם". הוצאתם משימה גדולה, אתם הולכים לישיבה, ובודקים מהנייד מה קרה. או מאשרים פעולות תוך כדי הליכה. או מנטרים build ארוך מהספה.

Claude Code כבר הגיע ל-29 מיליון התקנות יומיות ב-VS Code ולקצב הכנסות שנתי של 2.5 מיליארד דולר. Remote Control הוא עוד צעד לקראת הפיכת סוכני AI לכלי עבודה שנמצאים איתנו כל הזמן, לא רק כשאנחנו ליד מסך.

## הזווית המעשית: טיפים מהשטח

כמה דברים ששווה לדעת לפני שמתחילים:

**תנו שם לסשן.** לפני שמפעילים `/rc`, כתבו `/rename` ותנו שם ברור (למשל "refactor-auth-module"). ככה קל למצוא את הסשן ברשימה.

**היזהרו מ-QR code בצילומי מסך.** ה-URL של הסשן נותן גישה מלאה. אם אתם משתפים מסך או מצלמים סקרינשוט, וודאו שה-QR לא מופיע.

**הכירו את המגבלות.** סשן אחד בכל פעם, הטרמינל חייב להישאר פתוח, ואם הרשת נופלת ליותר מעשר דקות זה נגמר. תכננו בהתאם.

## השורה התחתונה

Remote Control הוא מהפיצ'רים שברגע שמתרגלים אליהם, אי אפשר לחזור אחורה. היכולת לשלוט על סוכן AI מקומי מכל מכשיר, בלי תשתית מיוחדת, בלי פתרונות עקומים, היא בדיוק מה שהיה חסר. עכשיו רק צריך שזה ייפתח גם למנויי Pro ול-Teams.
